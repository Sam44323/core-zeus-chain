Code for zeus-chain

This repo contains core zeus chain code

Features

  • Core Blockchain Implementation
  • POW methods
  • New block generation
  • Support for test-chain
  • Blockchain API support
  • Peers-to-Peers support, synchronization and addition
  • Consensus
  • Transaction pool
  • Key generation for chain using elliptic based cryptography
  • Reward for mining for miners
  • Multi-Chain Validation
  • WSS support
  • Wallet creation and transaction signing

Tech-stack

  • Typescript
  • Jest
  • Express
  • Crypto-JS
  • Nodemon
  • Elliptic
  • Winston
  • JSDOC
  • UUID
  • Socket.IO
  • Cors
  • Dotenv
  • Ts-node

Want to contribute? Hop on by forking the repo but surely go through the following scripts before starting the project

Scripts

Installing the dependencies

yarn

Running the watcher

yarn watch-proj

Running the logger(Optional)

note: Add a custom scripts in package.json with "dev-test" with nodemon runner before that

yarn dev-test

Running the tests

yarn test

Generating a documentation

yarn doc

Note:

You can add custom scripts to log other files
